# NoirScroll 🌑

**NoirScroll** is a high-performance, minimalist manga and webtoon reader designed for the ultimate reading experience. Built with a focus on speed, aesthetics, and user privacy, it provides a seamless interface to explore and read your favorite series.

![NoirScroll Preview](https://picsum.photos/seed/manga/1200/600)

## ✨ Features

- **🚀 Lightning Fast**: Optimized for quick loading and smooth scrolling.
- **🎨 Minimalist Design**: A clean, distraction-free interface that puts the content first.
- **📱 Responsive**: Perfect for both mobile and desktop reading.
- **📖 Multiple Reading Modes**: Support for both traditional Manga (paged) and Webtoon (vertical) formats.
- **🔍 Smart Search**: Easily find manga and specific chapters.
- **📥 Offline Reading**: Download chapters as high-quality PDFs for offline access.
- **🛡️ Privacy First**: No tracking, no data collection. Your history stays on your device.
- **🌓 Dark Mode**: Deep black themes optimized for OLED screens.

## 🛠️ Tech Stack

- **Frontend**: React 18 + TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **API**: AniList (Metadata) + Third-party Content Aggregators
- **PDF Generation**: jsPDF

## ⚖️ Legal Disclaimer

NoirScroll is a content aggregator. **We do not host, store, or upload any manga, webtoons, or images on our servers.** All content is provided by third-party sources via public APIs and web scraping. 

All intellectual property rights for the manga and webtoons belong to their respective creators and publishers. We encourage users to support official releases whenever possible.
